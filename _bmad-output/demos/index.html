Comparer les deux approches viewer

Même UI (pills) · 4 contrats PDF (yacht, Winnebago, jet ski, Ski-Doo) + 4 chats · seul le rendu PDF change. 

cd _bmad-output/demos
python3 generate-samples.py
python -m http.server 8765
Démo A — Natif

Look & feel navigateur

  • PDF : <iframe> (plugin / viewer Chrome, Firefox…)
  • JPEG : <img>
  • Pas de PDF.js · barre d’outils du navigateur
 Démo B — Librairies

Look & feel applicatif

  • PDF : PDF.js → canvas dans la page
  • JPEG : <img> (identique)
  • Zoom / pages intégrés à l’UI Desjardins

Contrats : Winnebago, jet ski, Ski-Doo + photo chat. Ouvrez les deux démos côte à côte sur un PDF.
